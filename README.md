## Live Demo

🚀 Streamlit App: https://plant-disease-detection-using-efficientnet-b0-marvpxhcqpeybsyr.streamlit.app

The application allows users to upload plant leaf images and instantly detect diseases using a trained EfficientNetB0 deep learning model. The system also provides disease descriptions, symptoms, causes, treatment recommendations, and prevention methods.


## Dataset

The dataset used for this project is available on Google Drive:

Dataset Link:
https://drive.google.com/drive/folders/1rERMftwTk-TgYfNa63w-1uDyiUio9rVo?usp=share_link

The dataset contains 14 classes of plant leaves from Apple, Corn, Grape, Strawberry, and Tomato plants, including healthy and diseased samples.
